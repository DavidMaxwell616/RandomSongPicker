24. That'll Never Happen No More

Besides Diddie Wa Diddie, this song is probably Blake's most well known. 
It's bouncy and full, using partial chords instead of single note fills.
The bass adds some syncopation for the offset "piano" sound. I play it 
capoed up two frets. I don't know if Blake used a capo, but the key he plays
in is A and it would be impossible to play these licks using 2nd position A
chords.
